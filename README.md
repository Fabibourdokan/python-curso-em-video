# 🐍 Python - Curso em Vídeo

Repositório dedicado aos meus estudos em Python, baseado no curso do professor Gustavo Guanabara, do canal Curso em Vídeo.

## 📚 Sobre o curso

O curso de Python 3 do Curso em Vídeo é um curso gratuito, voltado para iniciantes, que ensina desde os conceitos básicos de programação até estruturas mais avançadas da linguagem. :contentReference[oaicite:0]{index=0}  

Durante o curso, são propostos diversos exercícios práticos que ajudam a desenvolver o raciocínio lógico e a autonomia na programação.

## 🎯 Objetivo deste repositório

Este repositório tem como objetivo:

- Registrar minha evolução na programação
- Praticar lógica de programação com Python
- Armazenar soluções dos exercícios propostos
- Servir como material de consulta futura

## 🧠 Conteúdos abordados

- Variáveis e tipos de dados
- Operadores aritméticos e lógicos
- Estruturas condicionais (`if`, `elif`, `else`)
- Estruturas de repetição (`for`, `while`)
- Manipulação de strings
- Listas, tuplas e dicionários
- Funções
- Lógica de programação
